# 🗳️ Online Voting System

📌 Project Description
This is a web-based Online Voting System developed using Spring MVC, Hibernate, and MySQL.  
It allows users to register, login, and securely cast their votes online.

---
 🚀 Features
- User Registration & Login
- Secure Voting System
- Admin Dashboard
- Candidate Management
- Result Display

🛠️ Technologies Used
- Java
- Spring MVC
- Hibernate
- MySQL
- HTML, CSS, JavaScript

 📂 Project Structure
- src/main → Main source code
- pom.xml → Maven dependencies
- README.md → Project documentation

⚙️ How to Run
1. Clone the repository
2. Import project into Eclipse/IntelliJ
3. Configure MySQL database
4. Run on Apache Tomcat server
5. Open browser → http://localhost:8080
